Desktop Utility

输入一条 X / Twitter 链接，自动生成详情页截图

截图会直接保存到当前工具文件夹里的 screenshots/ 目录。 如果某些推文需要登录才能访问，可以勾选“显示浏览器”，在打开的浏览器里完成登录后再截图。

使用说明

  • 支持 `x.com`、`twitter.com`、`mobile.twitter.com` 的单条推文链接。
  • 默认优先截取推文详情页；如果详情页无法加载，会自动回退到嵌入卡片截图。
  • 截图文件会带上推文 ID 和时间戳，方便反复保存不覆盖。
